# Sankalp Team Next

Sankalp Team Next is a web application built using Next.js.  
The project is designed to provide a modern, responsive, and scalable platform with clean UI and optimized performance.

---

## Technologies Used

- Next.js
- React.js
- Tailwind CSS
- Node.js
- JavaScript / TypeScript
